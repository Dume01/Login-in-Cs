Logic Assignment

Objective

There are different applications which requires a logical framework to check the proof of natural deduction in Propositional Logic such as Flip. In this assignment we will be taking our first step to read a propositional logic formula and evaluate and print its truth table. Though Flip is written in Python Programming Language, this assignment will be done using C/C++.

Task 1: Write a function to convert the infix propositional logic expression into a prefix propositional logic expression.

Task 2: Write a function to convert the prefix expression into a rooted binary tree.

Task 3: Write a function to traverse the parse tree to output the infix expression back by in-order traversak of the parse tree.

Task 4: Write a function to compute the height of the parse tree.

Task 5: Write a function to evaluate the truth value of a propositional logic formula, given the truth values of each propositional atom by traversing the tree in a bottom up fashion.
