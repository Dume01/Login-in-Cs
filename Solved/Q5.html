Algorithm Implemented


Step 1: Firstly check if the tree is empty or not and return a value appropriately.

Step 2: The second corner case is if the tree contains only the root node. If such is the case return the value of the root node. This condition will also help in the case when durong the recursive call the function is at the leaf nodes.

Step 3: Call for the left subtree by root->left statement

Step 4: Call for the right subtree by root->right statement

Step 5: When an operator is encounterted perform the apt evaluation scheme as in natural deduction.

Step 6: Once the function as calculated things recursively return the final evaluation.
