Algorithm Implemented


For calculating the height of the Parse Tree we'll be needing the root node of a already made parse tree. We'll be performing a recursion to achieve the height.


Step 1: If the root is equal to null then return null.

Step 2: Call the left side of the subtree and save the value in an integer

Step 3: Call the right side of the subtree and save the value in an integer

Step 4: Return Maxiumum between integer value we got from the right and left traversal +1.
