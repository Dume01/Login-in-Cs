Algorithm Implemented


The following problem was broken by us into 2 parts. First part was converting the prefix expression into infix and the Second part was using both expressions to make the parse tree.

Part 1

We'll have make a structure of stack and use it.

Step 1: Read the expresion in prefix form from right to left that is in reverse order.

Step 2: If the character of the prefix is an operand then push it into a stack

Step 3: If the character from the prefix expression is an operator then pop 2 operands at the top of the stack and perform the following task

string=operand1+operator+operand2

push the resultant string into the stack

Step 4: Repeat the above steps until we reach the end of the Prefix expression

Step 5: Return the Peek of the stack once we the traverse the prefix. The peek will be the infix expression


Part 2

We'll make a structure of binary tree to implement this part of the problem.

Step 1: Pick an element from prefix expresion. Initialize a preorder index variable and increment it to pick the next element in a recursive call.

Step 2: Create a tree node say tN with the data as the picked element

Step 3: Find the picked element in the inorder expression we got from the part A. Let the index of that be inIndex.

Step 4: Call the function buildTree for elements before inIndex and make the built tree as a left subtree of tN.

Step 5: Call buildTree function for elements after inIndex and make the built treee as a right subtree of tN

Step 6: return the node tN.
